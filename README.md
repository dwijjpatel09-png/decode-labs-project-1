## 🚀 Project Title
### Zero-Shot & Few-Shot Data Extraction using Prompt Engineering

---

## 🎯 Objective
Design a structured AI prompt that converts **unstructured text into clean JSON output** using:
- Delimiters  
- Few-shot examples  
- Strict formatting rules  

This project demonstrates how prompt engineering can be used for **accurate data extraction in AI systems**.

---

## 📌 Problem Statement
Extract the following fields from messy/unstructured text:

- Name  
- Email  
- Phone  
- City  
- Occupation  

The output must always be returned in **valid JSON format only**.

---

## 🧠 Final Prompt Design

You are a Data Extraction AI.

### ⚙️ Rules:
1. Extract **Name, Email, Phone, City, and Occupation**.
2. Return **ONLY valid JSON**.
3. Do not add explanations or extra text.
4. If any field is missing, return `null`.
5. Follow formatting strictly as shown in examples.
